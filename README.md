# 🚀 AI Resume Analyzer + Job Matcher

A smart web application that analyzes resumes and provides insights like strengths, weaknesses, missing skills, and ATS score.

---

## 🔥 Features

* 📄 Upload Resume (PDF/TXT)
* 🧠 Automatic Resume Analysis
* ✅ Strength Detection
* ❌ Weakness Identification
* 📉 Missing Skills Suggestion
* 📊 ATS Score Calculation
* 💼 Job Matching Support (Optional)

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 🌐
* Machine Learning 🤖
* NLP (Basic)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 🔑 API Key Setup (Optional for AI version)

If using OpenAI:

1. Create API key from OpenAI
2. Add it in your code:

```python
client = OpenAI(api_key="YOUR_API_KEY")
```
